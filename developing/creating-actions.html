Accessing the Admin Interface

Now that you successfully setup Frapi (Both the admin and the frontend side) you must create your first action. As the Configuring Nginx and the Configuring Apache sections refer to the administration interface url as http://admin.frapi, we’ll use that URL in this example.

Firstly you must go to the admin interface and login. The default username is: admin and default password is: password.

Once this is done click on “Actions”

and you will be directed to this screen

and as the arrow suggests, click on “Add new Action”.
